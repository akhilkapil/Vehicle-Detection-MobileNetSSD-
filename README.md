# Vehicle-Detection-MobileNetSSD

A Vehicle count detection model, that can read real-time moving objects and define their class. The model use a DNN to detect and classify different objects that may be in any given frame. The model we are using is a caffe version of model called MobileNet-SSD which uses a hybrid of a framework from Google called MobileNet and another framework called Single Shot Detector (SSD) MultiBox.

The output has been saved by the name 'Output_mobilenet-ssd.mp4'.
